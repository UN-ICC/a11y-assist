# act-rules-query

Programmatic access to W3C [Accessibility Conformance Testing (ACT) Rules](https://act-rules.github.io/), loaded verbatim from snapshotted YAML/Markdown — each rule mapped to the WCAG Success Criteria it covers. Part of [a11y-assist](https://github.com/UN-ICC/a11y-assist).

📖 **Docs:** <https://un-icc.github.io/a11y-assist/packages/act-rules-query/>

## Install

```sh
npm install act-rules-query
```
