# apg-query

Programmatic, **verbatim** access to the [W3C ARIA Authoring Practices Guide](https://www.w3.org/WAI/ARIA/apg/) (APG) patterns — analogous to [`aria-query`](https://www.npmjs.com/package/aria-query), but for APG. Part of [a11y-assist](https://github.com/UN-ICC/a11y-assist).

📖 **Docs:** <https://un-icc.github.io/a11y-assist/packages/apg-query/>

## Install

```sh
npm install apg-query
```
