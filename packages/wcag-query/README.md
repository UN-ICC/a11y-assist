# wcag-query

Programmatic, **verbatim** access to [WCAG 2.2](https://www.w3.org/TR/WCAG22/) Success Criteria, Sufficient Techniques, and Failures. Part of [a11y-assist](https://github.com/UN-ICC/a11y-assist).

📖 **Docs:** <https://un-icc.github.io/a11y-assist/packages/wcag-query/>

## Install

```sh
npm install wcag-query
```
