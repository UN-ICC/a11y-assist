WCAG 2.2 · Level A

3.1.1 Language of Page

Success Criterion

The default human language of each web page can be programmatically determined.

Understanding 3.1.1 on W3C

Sufficient techniques

  • H57 Using the language attribute on the HTML element
  • PDF16 Setting the default language using the /Lang entry in the document catalog of a PDF document
  • PDF19 Specifying the language for a passage or phrase with the Lang entry in PDF documents

ACT rules covering this SC mechanical

  • 5b7ae0 HTML page lang and xml:lang attributes have matching values
  • b5c3f8 HTML page has lang attribute
  • bf051a HTML page lang attribute has valid language tag
  • ucwvc8 HTML page language subtag matches default language
