WCAG 2.2 · Level AAA

1.4.9 Images of Text (No Exception)

Success Criterion

Images of text are only used for pure decoration or where a particular presentation of text is essential to the information being conveyed. Note Logotypes (text that is part of a logo or brand name) are considered essential.

Understanding 1.4.9 on W3C

Sufficient techniques

  • C22 Using CSS to control visual presentation of text
  • C30 Using CSS to replace text with images of text and providing user interface controls to switch
  • G140 Separating information and structure from presentation to enable different presentations
  • PDF7 Performing OCR on a scanned PDF document to provide actual text
