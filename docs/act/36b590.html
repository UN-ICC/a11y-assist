ACT Rule · atomic

Error message describes invalid form field value

id: 36b590

https://act-rules.github.io/rules/36b590

Description

This rule checks that text error messages provided when the user completes a form field with invalid values or using an invalid format, identify the cause of the error or how to fix the error.

WCAG Success Criteria covered

  • 3.3.1

Applicability verbatim from upstream

This rule applies to each [HTML element][] that has one of the following [semantic roles][semantic role]:

- `checkbox`,
- `combobox`,
- `listbox`,
- `menuitemcheckbox`,
- `menuitemradio`,
- `radio`,
- `searchbox`,
- `slider`,
- `spinbutton`,
- `switch` or
- `textbox`.

Patterns matched by this rule

  • spinbutton
