ACT Rule · composite

Video element visual content has strict accessible alternative

id: 1ec09b

https://act-rules.github.io/rules/1ec09b

Description

This rule checks that `video` elements with audio have audio description.

WCAG Success Criteria covered

  • 1.2.5

Applicability verbatim from upstream

This rule applies to every [non-streaming](#non-streaming-media-element) `video` element that is [visible][], where the video contains audio.
