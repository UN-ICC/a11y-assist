Platform binding

Native HTML elements from aria-query

Elements that natively carry one of this pattern's ARIA roles. Using these gives you the role for free — no role="…" attribute needed.

  • <button> · implicit role button
  • <input[type=button]> · implicit role button
  • <input[type=image]> · implicit role button
  • <input[type=reset]> · implicit role button
  • <input[type=submit]> · implicit role button
