Platform binding

Native HTML elements from aria-query

Elements that natively carry one of this pattern's ARIA roles. Using these gives you the role for free — no role="…" attribute needed.

  • <input[list][type=email]> · implicit role combobox
  • <input[list][type=search]> · implicit role combobox
  • <input[list][type=tel]> · implicit role combobox
  • <input[list][type=text]> · implicit role combobox
  • <input[list][type=url]> · implicit role combobox
  • <select[!multiple][!size]> · implicit role combobox · the multiple attribute is not set and the size attribute does not have a value greater than 1
  • <dialog> · implicit role dialog
  • <datalist> · implicit role listbox
  • <select[multiple]> · implicit role listbox
  • <select[size]> · implicit role listbox · the size attribute value is greater than 1
