Platform binding

Native HTML elements from aria-query

Elements that natively carry one of this pattern's ARIA roles. Using these gives you the role for free — no role="…" attribute needed.

  • <th> · implicit role columnheader
  • <th[scope=col]> · implicit role columnheader
  • <th[scope=colgroup]> · implicit role columnheader
  • <td> · implicit role gridcell · ancestor table element has grid role, ancestor table element has treegrid role
  • <tr> · implicit role row
  • <tbody> · implicit role rowgroup
  • <tfoot> · implicit role rowgroup
  • <thead> · implicit role rowgroup
  • <th[scope=row]> · implicit role rowheader
  • <th[scope=rowgroup]> · implicit role rowheader
