Platform binding

Native HTML elements from aria-query

Elements that natively carry one of this pattern's ARIA roles. Using these gives you the role for free — no role="…" attribute needed.

  • <input[!list][!type]> · implicit role textbox · the list attribute is not set
  • <input[!list][type=email]> · implicit role textbox · the list attribute is not set
  • <input[!list][type=tel]> · implicit role textbox · the list attribute is not set
  • <input[!list][type=text]> · implicit role textbox · the list attribute is not set
  • <input[!list][type=url]> · implicit role textbox · the list attribute is not set
  • <textarea> · implicit role textbox
